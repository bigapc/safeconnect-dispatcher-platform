# Architecture Guide

## Frontend

- Next.js 15 App Router
- Tailwind CSS for design system implementation
- React Query for data caching and server state
- Axios for API calls

## Backend

- NestJS with modular architecture
- Prisma ORM on PostgreSQL
- JWT auth with refresh token cookies
- Socket.IO for realtime courier updates

## Data

Core entities include Organization, User, Dispatcher, Courier, Assignment, Incident, Notification, AuditLog, Role, Permission, Subscription, Invoice, Webhook, GPSLocation, OrganizationSettings, and AnalyticsSnapshot.
